## [1.0.0] - 2026-04-05

### Added
- Initial release of NWO Cardiac SDK
- Dual-modal cardiac biometrics (WiFi CSI + BLE Wearables)
- Multi-modal fusion engine with liveness detection
- Four market modules:
  - Access Control: SecureEntry for facility access
  - Wellness & Fitness: FitnessTracker for gym/studio tracking
  - Clinical Research: RemoteMonitor for patient monitoring
  - Robotics: HumanAwareRobot for human-robot collaboration
- Support for Apple Watch and Polar H10 wearables
- WiFi CSI presence detection and heart rate extraction
- REST API endpoint (api-cardiac.php)
- CLI interface (nwo-cardiac command)
- Comprehensive documentation and examples

### Features
- Identity verification with 97%+ confidence
- Contactless liveness detection
- Continuous authentication
- Clinical-grade anomaly detection
- Anti-spoofing protection
- GDPR/HIPAA compliant design
